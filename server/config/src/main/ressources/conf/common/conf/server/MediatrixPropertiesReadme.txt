#### Erläuterungen zu property Einstellungen ####


nouseindex=1
    Betrifft Ihre Anfrage mit der Vorgangsnummer: 2131528 (Bei Rückfragen bitte immer mit angeben)
    Mit diesem Parameter wird die Datenbank gezwungen einen gewissen Index innerhalb der MailInbox zu verwenden. Dies muss aber je nach Datenbanksystem nicht immer vorteilhaft sein. Bei einem Oracle System wie bei Sky im Einsatz würden wir dazu raten dies Parameter auf 1 zu setzen um der Datenbank die Entscheidung über die Nutzung des Index zu überlassen.
